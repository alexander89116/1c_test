# Задача про курсы
Простое переборное решение.
## Формат ввода
В начале первые шесть строк - уровни, начиная с нулевого. В каждой строке первое число - количество курсов на уровне, далее номера курсов.
```
1 0 // на первом уровне один курс под номером 0
1 1 // второй уровень
1 2
3 3 6 7
1 4
1 5
```
Дальше идёт N строк, где N - количество курсов. В каждой строке указаны зависимости, первое число - количество групп. В каждой группе - первое число - количество курсов, далее их перечисление
```
3 2 3 4 1 5 1 6 // 3 группы, [3, 4], [5], [6], т е для этого курса надо закончить 3 ИЛИ 4, 5 и 6
```
Дальше строка с приоритеми для второго уровня
```
3 2 1 // количество курсов на втором уровне - 3
```
Строка из 5 чисел: обязательный курс с 5ого уровня, 3 обязательныйх со 2ого уровня, минимальное количество курсов, которые надо закрыть
